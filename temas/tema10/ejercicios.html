Ejercicio 1

Operaciones Básicas de Conjuntos

Dados los conjuntos:

  • A = {1, 2, 3, 4, 5}
  • B = {3, 4, 5, 6, 7}
  • C = {2, 4, 6, 8}

Calcula las siguientes operaciones:

  1. A ∪ B
  2. A ∩ B
  3. A - B
  4. B - A
  5. (A ∪ B) ∩ C
Pista

Recuerda que:

  • La unión (∪) incluye todos los elementos que están en A o en B
  • La intersección (∩) incluye solo los elementos que están en A y en B
  • La diferencia (A - B) incluye los elementos que están en A pero no en B
Solución
  1. A ∪ B = {1, 2, 3, 4, 5, 6, 7}
  2. A ∩ B = {3, 4, 5}
  3. A - B = {1, 2}
  4. B - A = {6, 7}
  5. (A ∪ B) ∩ C = {1, 2, 3, 4, 5, 6, 7} ∩ {2, 4, 6, 8} = {2, 4, 6}
Ejercicio 2

Conjunto Potencia

Dado el conjunto A = {a, b}, encuentra P(A) (el conjunto potencia de A).

¿Cuántos elementos tiene P(A)?

Pista

El conjunto potencia P(A) incluye todos los subconjuntos de A, incluyendo el conjunto vacío ∅ y el conjunto A completo.

Solución

P(A) = {∅, {a}, {b}, {a, b}}

P(A) tiene 4 elementos.

En general, si un conjunto tiene n elementos, su conjunto potencia tiene 2ⁿ elementos.

Ejercicio 3

Producto Cartesiano

Dados los conjuntos:

  • A = {1, 2}
  • B = {x, y}

Calcula A × B y B × A. ¿Son iguales?

Pista

El producto cartesiano A × B incluye todos los pares ordenados donde el primer elemento pertenece a A y el segundo a B.

Solución

A × B = {(1, x), (1, y), (2, x), (2, y)}

B × A = {(x, 1), (x, 2), (y, 1), (y, 2)}

No, A × B ≠ B × A. El producto cartesiano no es conmutativo.

Ejercicio 4

Relaciones y Funciones

Dado el conjunto A = {1, 2, 3, 4}, considera las siguientes relaciones:

  • R₁ = {(1, 2), (2, 3), (3, 4)}
  • R₂ = {(1, 1), (2, 2), (3, 3), (4, 4)}
  • R₃ = {(1, 2), (1, 3), (2, 4)}

¿Cuáles de estas relaciones son funciones? Justifica tu respuesta.

Pista

Una función es una relación donde cada elemento del dominio tiene exactamente una imagen (univocidad por la derecha).

Solución
  • R₁: Es función. Cada elemento del dominio {1, 2, 3} tiene exactamente una imagen.
  • R₂: Es función. Es la función identidad.
  • R₃: No es función. El elemento 1 tiene dos imágenes diferentes (2 y 3).
