Semántica de la Lógica de Predicados

Comprensión de la semántica estándar para la lógica de predicados y su evaluación

Progreso del Tema 0%

Introducción y Objetivos

Una de las principales dificultades que tiene la presentación de una semántica para la lógica de predicados es que hay que tener en cuenta el valor semántico de todas las fórmulas bien formadas, ya sean estas sentencias o no.

Conceptos Clave

Además de la noción de posible realización, que es la estructura que proporciona el valor semántico de las sentencias, necesitaremos introducir también el concepto de asignación.

Introducción a la Semántica de Predicados

Concepto de Asignación

Dada una posible realización A = <A, ƒ> y una asignación J para la lógica de predicados, donde:

  • Si a es una constante de individuo de la lógica de predicados, entonces ƒ(a) ∈ A.
  • Si R es un predicado n-ádico de la lógica de predicados, entonces ƒ(R) ⊆ Aⁿ.
  • J: VAR → A, siendo VAR el conjunto de variables de individuo de la lógica de predicados.

Semántica Estándar

La semántica estándar para la lógica de predicados queda definida por:

  • Los valores semánticos de expresiones básicas (no lógicas)
  • Las reglas semánticas para fbf
  • Las definiciones para la evaluación semántica de fbf cuando el valor veritativo correspondiente es indiferente de qué asignación se seleccione

Valores Semánticos de Expresiones Básicas

Constantes de Individuo

Si a es una constante de individuo de la lógica de predicados, entonces:

[a]ᴬⱼ = ƒ(a)

Predicados n-ádicos

Si R es un predicado n-ádico de la lógica de predicados, entonces:

ƒ(R) ⊆ Aⁿ

Variables de Individuo

Si x es una variable de individuo de la lógica de predicados, entonces:

[x]ᴬⱼ = J(x)

Reglas Semánticas para FBF

RSF1: Predicados

Dado un predicado n-ádico cualquiera del lenguaje, R, y una sucesión cualquiera de n términos del lenguaje, τ₁, ..., τₙ:

[Rτ₁, ..., τₙ]ᴬⱼ = 1 syss <[τ₁]ᴬⱼ, [τ₂]ᴬⱼ, ..., [τₙ]ᴬⱼ> ∈ [R]ᴬⱼ

Cuando R es monario, se dice que:

[Rτ₁]ᴬⱼ = 1 syss [τ₁]ᴬⱼ ∈ [R]ᴬⱼ

RSF2: Negación

Si A es una fbf de la lógica de predicados, entonces:

[¬A]ᴬⱼ = 1 syss [A]ᴬⱼ = 0

RSF3: Conectivos Lógicos

Si A y B son fbf de la lógica de predicados, entonces:

  • Disyunción: [A ∨ B]ᴬⱼ = 1 syss [A]ᴬⱼ = 1 o [B]ᴬⱼ = 1
  • Conjunción: [A ∧ B]ᴬⱼ = 1 syss [A]ᴬⱼ = 1 y [B]ᴬⱼ = 1
  • Implicación: [A → B]ᴬⱼ = 1 syss [A]ᴬⱼ = 0 o [B]ᴬⱼ = 1

RSF4: Cuantificadores

Si A es una fbf de la lógica de predicados, entonces:

Cuantificador Existencial

[∃x A]ᴬⱼ = 1 syss para al menos un elemento α del dominio A se cumple que [A]ᴬⱼ = 1

donde J' es una asignación en todo igual a J salvo, posiblemente en que asigna α a la variable de individuo x.

Cuantificador Universal

[∀x A]ᴬⱼ = 1 syss para todo elemento α del dominio A se cumple que [A]ᴬⱼ = 1

donde J' es una asignación en todo igual a J salvo, posiblemente en que asigna α a la variable x.

Definiciones para la Evaluación Semántica

Verdad en una Realización

Si A es una fbf de la lógica de predicados, entonces:

[A]ᴬ = 1 por definición syss [A]ᴬⱼ = 1 para toda asignación J

Falsedad en una Realización

Si A es una fbf de la lógica de predicados, entonces:

[A]ᴬ = 0 por definición syss [A]ᴬⱼ = 0 para toda asignación J

Ejemplos Prácticos

Ejemplo de Evaluación

Dada la posible realización A y asignación J:

  • A = {1, 2, 3, 4, 5, 6, 7, 8, 9, 0}
  • ƒ(a) = 3, ƒ(b) = 5, ƒ(c) = 9, ƒ(d) = 0
  • ƒ(S) = {3, 6, 7, 8}
  • ƒ(R) = {<0,5>, <9,4>, <9,1>}
  • J(x) = 2, J(y) = 4, J(z) = 6

Evaluación de Sd

Sd

Proceso: [Sd]ᴬⱼ = 1 syss [d]ᴬⱼ ∈ [S]ᴬⱼ

[d]ᴬⱼ = ƒ(d) = 0

[S]ᴬⱼ = ƒ(S) = {3, 6, 7, 8}

Resultado: 0 ∉ {3, 6, 7, 8}, por tanto [Sd]ᴬⱼ = 0

Evaluación de Rda

Rda

Proceso: [Rda]ᴬⱼ = 1 syss <[d]ᴬⱼ, [a]ᴬⱼ> ∈ [R]ᴬⱼ

[d]ᴬⱼ = ƒ(d) = 0, [a]ᴬⱼ = ƒ(a) = 3

[R]ᴬⱼ = ƒ(R) = {<0,5>, <9,4>, <9,1>}

Resultado: <0,3> ∉ {<0,5>, <9,4>, <9,1>}, por tanto [Rda]ᴬⱼ = 0

Relación con la Teoría de Conjuntos

La semántica de la lógica de predicados se fundamenta en la teoría de conjuntos:

Individuos

Los individuos en la lógica de predicados corresponden a elementos de conjuntos en la teoría de conjuntos.

Predicados Monádicos

Los predicados monádicos corresponden a conjuntos en la teoría de conjuntos.

Predicados Poliádicos

Los predicados poliádicos corresponden a relaciones en la teoría de conjuntos.
