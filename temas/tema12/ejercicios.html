Ejercicios Prácticos

Tema 12: Aplicaciones de la Lógica de Predicados - Programación Lógica

Ejercicios Básicos

1
Fácil

Base de Hechos

Crea una base de hechos en Prolog que represente la siguiente información:

  • Juan es padre de María
  • María es madre de Pedro
  • Ana es hermana de Juan
  • Carlos es hijo de Ana

Solución:

% Base de hechos familiares
padre(juan, maria).
madre(maria, pedro).
hermana(ana, juan).
hijo(carlos, ana).
2
Fácil

Consultas Básicas

Dada la siguiente base de hechos:

animal(perro).
animal(gato).
animal(pajaro).
domestico(perro).
domestico(gato).

¿Qué consultas harías para obtener:

  • Todos los animales
  • Todos los animales domésticos
  • Verificar si el perro es doméstico

Solución:

% Obtener todos los animales
?- animal(X).

% Obtener todos los animales domésticos
?- domestico(X).

% Verificar si el perro es doméstico
?- domestico(perro).

Ejercicios Intermedios

3
Medio

Reglas de Parentesco

Usando la base de hechos del ejercicio 1, define reglas para:

  • abuelo(X, Y): X es abuelo de Y
  • tio(X, Y): X es tío de Y
  • primo(X, Y): X es primo de Y

Solución:

% Un abuelo es padre del padre o madre
abuelo(X, Y) :- padre(X, Z), padre(Z, Y).
abuelo(X, Y) :- padre(X, Z), madre(Z, Y).

% Un tío es hermano del padre o madre
tio(X, Y) :- hermano(X, Z), padre(Z, Y).
tio(X, Y) :- hermano(X, Z), madre(Z, Y).

% Un primo es hijo de hermanos
primo(X, Y) :- hijo(X, A), hijo(Y, B), hermano(A, B).
4
Medio

Sistema de Clasificación

Crea un sistema de clasificación de animales con las siguientes reglas:

  • Un mamífero es un animal que tiene pelo
  • Un ave es un animal que tiene plumas
  • Un depredador es un animal que come carne
  • Un herbívoro es un animal que come plantas

Define hechos y reglas para clasificar diferentes animales.

Solución:

% Hechos básicos
tiene_pelo(leon).
tiene_pelo(vaca).
tiene_plumas(aguila).
come_carne(leon).
come_plantas(vaca).

% Reglas de clasificación
mamifero(X) :- tiene_pelo(X).
ave(X) :- tiene_plumas(X).
depredador(X) :- come_carne(X).
herbivoro(X) :- come_plantas(X).

Ejercicios de Unificación y Backtracking

5
Medio

Unificación de Términos

Dados los siguientes pares de términos, determina si se unifican y cuáles serían las sustituciones:

  • padre(juan, X) y padre(Y, maria)
  • [a, b, c] y [X | Y]
  • f(X, Y) y f(a, b)
  • g(X, X) y g(Y, Z)

Solución:

% 1. padre(juan, X) y padre(Y, maria)
% Sí se unifican: X = maria, Y = juan

% 2. [a, b, c] y [X | Y]
% Sí se unifican: X = a, Y = [b, c]

% 3. f(X, Y) y f(a, b)
% Sí se unifican: X = a, Y = b

% 4. g(X, X) y g(Y, Z)
% No se unifican (X no puede ser igual a Y y Z simultáneamente)
6
Medio

Simulación de Backtracking

Dada la siguiente base de hechos y reglas:

padre(juan, maria).
padre(juan, pedro).
padre(carlos, ana).
abuelo(X, Y) :- padre(X, Z), padre(Z, Y).

¿Qué respuestas obtendrías para la consulta ?- abuelo(X, Y).?

Solución:

La consulta abuelo(X, Y) no tendría respuestas porque no hay relaciones de abuelo-nieto en la base de hechos. Para que haya abuelos, necesitaríamos hechos como:

padre(maria, luis).
padre(pedro, sofia).

Entonces sí tendríamos: X = juan, Y = luis y X = juan, Y = sofia

Ejercicios con Listas

7
Medio

Predicados con Listas

Implementa los siguientes predicados para trabajar con listas:

  • suma_lista(Lista, Suma): Suma todos los elementos de una lista
  • maximo_lista(Lista, Maximo): Encuentra el elemento máximo
  • invertir(Lista, Invertida): Invierte el orden de los elementos

Solución:

% Suma de lista
suma_lista([], 0).
suma_lista([X | T], S) :-
    suma_lista(T, S1),
    S is X + S1.

% Máximo de lista
maximo_lista([X], X).
maximo_lista([X | T], M) :-
    maximo_lista(T, M1),
    X > M1, M is X.
maximo_lista([X | T], M) :-
    maximo_lista(T, M).

% Invertir lista
invertir([], []).
invertir([X | T], R) :-
    invertir(T, R1),
    append(R1, [X], R).

Ejercicios de Recursión

8
Medio

Recursión Matemática

Implementa los siguientes predicados recursivos:

  • fibonacci(N, F): Calcula el N-ésimo número de Fibonacci
  • potencia(Base, Expo, Resultado): Calcula Base^Expo
  • suma_numeros(N, Suma): Suma de 1 a N

Solución:

% Fibonacci
fibonacci(0, 0).
fibonacci(1, 1).
fibonacci(N, F) :-
    N > 1,
    N1 is N - 1,
    N2 is N - 2,
    fibonacci(N1, F1),
    fibonacci(N2, F2),
    F is F1 + F2.

% Potencia
potencia(_, 0, 1).
potencia(B, E, R) :-
    E > 0,
    E1 is E - 1,
    potencia(B, E1, R1),
    R is B * R1.

% Suma de números
suma_numeros(0, 0).
suma_numeros(N, S) :-
    N > 0,
    N1 is N - 1,
    suma_numeros(N1, S1),
    S is N + S1.

Ejercicios Avanzados

9
Difícil

Uso del Corte

Implementa un predicado maximo(X, Y, Z) que devuelva el máximo entre X e Y en Z, usando el corte para evitar respuestas múltiples.

Solución:

maximo(X, Y, X) :- X >= Y, !.
maximo(X, Y, Y).

Explicación: El corte (!) evita que se evalúe la segunda regla si la primera se cumple, garantizando una sola respuesta.

Sistema de Recomendaciones

Crea un sistema de recomendaciones de libros basado en géneros y preferencias del usuario. Incluye reglas para:

  • Recomendar libros por género
  • Recomendar libros por autor
  • Recomendar libros similares

Solución:

% Base de hechos
libro(cien_anos_soledad, garcia_marquez, realismo_magico).
libro(1984, orwell, ciencia_ficcion).
prefiere(usuario1, realismo_magico).

% Reglas de recomendación
recomendar_por_genero(Usuario, Libro) :-
    prefiere(Usuario, Genero),
    libro(Libro, _, Genero).

recomendar_por_autor(Usuario, Libro) :-
    ha_leido(Usuario, LibroAnterior),
    libro(LibroAnterior, Autor, _),
    libro(Libro, Autor, _).
