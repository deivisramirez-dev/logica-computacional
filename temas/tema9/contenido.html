Estrategias de Deducción para la Lógica de Predicados

Técnicas y reglas para realizar deducciones en lógica de predicados

Progreso del Tema 0%

Introducción y Objetivos

En este tema se explican las bases de la deducción en lógica de predicados. Aprenderemos a realizar deducciones utilizando las reglas de inferencia específicas para cuantificadores y predicados.

Objetivo Principal

Dominar las técnicas de deducción para demostrar la validez de argumentos en lógica de predicados, utilizando correctamente las reglas de introducción y eliminación de cuantificadores.

Introducción a la Deducción en Lógica de Predicados

La deducción en lógica de predicados extiende las reglas de la lógica proposicional con reglas específicas para manejar cuantificadores. Estas reglas nos permiten introducir y eliminar cuantificadores de manera controlada.

Las reglas de deducción para lógica de predicados incluyen:

  • Reglas de eliminación de cuantificadores: Para extraer información de fórmulas cuantificadas
  • Reglas de introducción de cuantificadores: Para crear nuevas fórmulas cuantificadas
  • Reglas de sustitución: Para reemplazar variables por términos

Reglas de Eliminación de Cuantificadores

Eliminación del Cuantificador Universal (∀E)

Si tenemos una fórmula de la forma ∀xP(x), podemos inferir P(t) para cualquier término t.

∀xP(x) ⊢ P(t)

Condiciones

  • t puede ser cualquier término (constante, variable o función)
  • No hay restricciones especiales para esta regla

Ejemplo

1 ∀x(P(x) → Q(x)) Premisa
2 P(a) → Q(a) ∀E, 1

Eliminación del Cuantificador Existencial (∃E)

Si tenemos ∃xP(x) y podemos derivar Q asumiendo P(c) para una constante nueva c, entonces podemos inferir Q.

∃xP(x), P(c) ⊢ Q → Q

Condiciones

  • c debe ser una constante nueva (no usada antes en la deducción)
  • c no puede aparecer en Q
  • c no puede aparecer en ninguna premisa anterior

Ejemplo

1 ∃xP(x) Premisa
2 P(c) Supuesto para ∃E
3 Q Derivado de 2
4 Q ∃E, 1, 2-3

Reglas de Introducción de Cuantificadores

Introducción del Cuantificador Universal (∀I)

Si podemos derivar P(c) para una constante arbitraria c, entonces podemos inferir ∀xP(x).

P(c) ⊢ ∀xP(x)

Condiciones

  • c debe ser una constante arbitraria (no usada en premisas)
  • c no puede aparecer en ninguna premisa anterior
  • c no puede aparecer en la conclusión

Ejemplo

1 P(c) Premisa
2 ∀xP(x) ∀I, 1

Introducción del Cuantificador Existencial (∃I)

Si tenemos P(t) para algún término t, podemos inferir ∃xP(x).

P(t) ⊢ ∃xP(x)

Condiciones

  • t puede ser cualquier término
  • No hay restricciones especiales para esta regla

Ejemplo

1 P(a) Premisa
2 ∃xP(x) ∃I, 1

Estrategias para Deducciones

Para realizar deducciones efectivas en lógica de predicados, sigue estos pasos:

1

Identificar la Estructura

Analiza las premisas y la conclusión para identificar qué cuantificadores y predicados están involucrados.

2

Eliminar Cuantificadores

Usa las reglas de eliminación para extraer información de las fórmulas cuantificadas.

3

Aplicar Lógica Proposicional

Usa las reglas de la lógica proposicional para manipular las fórmulas sin cuantificadores.

4

Introducir Cuantificadores

Usa las reglas de introducción para crear la fórmula cuantificada deseada.

Ejemplos de Deducciones

Ejemplo 1: Modus Ponens con Cuantificadores

Premisas:

  • ∀x(P(x) → Q(x))
  • P(a)

Conclusión: Q(a)

1 ∀x(P(x) → Q(x)) Premisa
2 P(a) Premisa
3 P(a) → Q(a) ∀E, 1
4 Q(a) →E, 2, 3

Ejemplo 2: Generalización Universal

Premisa: P(c) (donde c es arbitraria)

Conclusión: ∀xP(x)

1 P(c) Premisa
2 ∀xP(x) ∀I, 1

Ejemplo 3: Eliminación Existencial

Premisas:

  • ∃xP(x)
  • ∀x(P(x) → Q(x))

Conclusión: ∃xQ(x)

1 ∃xP(x) Premisa
2 ∀x(P(x) → Q(x)) Premisa
3 P(c) Supuesto para ∃E
4 P(c) → Q(c) ∀E, 2
5 Q(c) →E, 3, 4
6 ∃xQ(x) ∃I, 5
7 ∃xQ(x) ∃E, 1, 3-6

Errores Comunes en Deducciones

Error 1: Violar Restricciones de ∀I

Incorrecto: Usar ∀I con una constante que aparece en premisas anteriores.

Correcto: Usar ∀I solo con constantes arbitrarias que no aparecen en premisas.

Error 2: Violar Restricciones de ∃E

Incorrecto: Usar ∃E con una constante que ya aparece en la conclusión.

Correcto: Usar ∃E con una constante completamente nueva.

Error 3: Confundir Orden de Aplicación

Incorrecto: Aplicar reglas en orden incorrecto.

Correcto: Seguir el orden lógico: eliminar cuantificadores, aplicar lógica proposicional, introducir cuantificadores.
