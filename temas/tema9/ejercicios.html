Ejercicios de Deducción

Practica las técnicas de deducción en lógica de predicados

Instrucciones

Realiza las siguientes deducciones utilizando las reglas de inferencia para lógica de predicados. Sigue los pasos de deducción explicados en el contenido del tema.

Reglas disponibles:

  • ∀E (Eliminación Universal): ∀xP(x) ⊢ P(t)
  • ∃E (Eliminación Existencial): ∃xP(x), P(c) ⊢ Q → Q
  • ∀I (Introducción Universal): P(c) ⊢ ∀xP(x)
  • ∃I (Introducción Existencial): P(t) ⊢ ∃xP(x)
  • Reglas proposicionales: MP, MT, HS, DS, etc.

Pasos a seguir:

  • Identificar la estructura de premisas y conclusión
  • Eliminar cuantificadores usando ∀E y ∃E
  • Aplicar lógica proposicional para manipular fórmulas
  • Introducir cuantificadores usando ∀I y ∃I
1

Ejercicio 1

Premisas:
  • ∀x(P(x) → Q(x))
  • P(a)
Conclusión: Q(a)
1
2
3
4

Solución

1. ∀x(P(x) → Q(x)) Premisa
2. P(a) Premisa
3. P(a) → Q(a) ∀E, 1
4. Q(a) MP, 2, 3

Explicación: Primero eliminamos el cuantificador universal de la primera premisa para obtener P(a) → Q(a). Luego aplicamos Modus Ponens con P(a) para obtener Q(a).

2

Ejercicio 2

Premisas:
  • P(c) (donde c es arbitraria)
Conclusión: ∀xP(x)
1
2

Solución

1. P(c) Premisa
2. ∀xP(x) ∀I, 1

Explicación: Como c es una constante arbitraria (no aparece en premisas), podemos aplicar la regla de introducción universal para generalizar a ∀xP(x).

Ejercicio 3

Premisas:
  • P(a)
Conclusión: ∃xP(x)
1
2

Solución

1. P(a) Premisa
2. ∃xP(x) ∃I, 1

Explicación: Aplicamos directamente la regla de introducción existencial, ya que tenemos P(a) y podemos inferir ∃xP(x).

Ejercicio 4

Premisas:
  • ∃xP(x)
  • ∀x(P(x) → Q(x))
Conclusión: ∃xQ(x)

Solución

1. ∃xP(x) Premisa
2. ∀x(P(x) → Q(x)) Premisa
3. P(c) Supuesto para ∃E
4. P(c) → Q(c) ∀E, 2
5. Q(c) MP, 3, 4
6. ∃xQ(x) ∃I, 5
7. ∃xQ(x) ∃E, 1, 3-6

Explicación: Usamos eliminación existencial para asumir P(c), luego aplicamos ∀E para obtener P(c) → Q(c), aplicamos MP para obtener Q(c), introducimos ∃xQ(x) con ∃I, y finalmente aplicamos ∃E para concluir ∃xQ(x).

Ejercicio 5

Premisas:
  • ∀x(P(x) → Q(x))
  • ∀x(Q(x) → R(x))
Conclusión: ∀x(P(x) → R(x))
1
2
3
4

Solución

1. ∀x(P(x) → Q(x)) Premisa
2. ∀x(Q(x) → R(x)) Premisa
3. P(c) → Q(c) ∀E, 1
4. Q(c) → R(c) ∀E, 2
5. P(c) → R(c) HS, 3, 4
6. ∀x(P(x) → R(x)) ∀I, 5

Explicación: Eliminamos los cuantificadores universales para obtener P(c) → Q(c) y Q(c) → R(c), aplicamos Silogismo Hipotético para obtener P(c) → R(c), y finalmente aplicamos ∀I para generalizar a ∀x(P(x) → R(x)).
